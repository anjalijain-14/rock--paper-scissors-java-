This is a simple Java console-based Rock Paper Scissors game.

## How to Play
1. Run the program in Java IDE (like IntelliJ or Eclipse)
2. Choose one option: Rock, Paper, or Scissors
3. Computer randomly selects its choice
4. Winner is decided based on rules:
   - Rock beats Scissors
   - Scissors beats Paper
   - Paper beats Rock

## Concepts Used
- Conditional Statements (if-else)
- Loops
- User Input with Scanner class
